# 翻译说明

欢迎你对我们翻译计划感兴趣，请在翻译前阅读本文件。

## 注意事项
1. 翻译需要lbc分支的基础上新建直接的分支上进行，校队后合并到lbc分支。
2. 翻译应该在意思相同的情况下使用中文习惯的表达方式。
3. 翻译时一定要经常链接之后可以访问。
4. 翻译时不要修改原文的格式（结构）
5. 认领翻译后，以及更新[进度](trans-progress.md)。

## 本地运行
进入website 目录，安装依赖：
```
yarn install
```

构建文档：
```
yarn start
```
